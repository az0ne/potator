incomingCallback(spore_string)[source]¶

Callback when a spore is received from a remote node

Parameters:spore_string (str) – The spore encoded string

The spore is decoded from the spore_string using protobuf. spore is then passed into potator.network_dispatcher.NetworkDispatcher.handleDispatch().

Depending on whether the spore contains IP or OURP data, it will then be passed into the OURP module or Tuntap module.

outgoingCallback(packet)[source]¶

Callback when an outgoing IP packet is received from Tuntap

Parameters:packet – The IP packet

The packet is loaded into a Spore object and encoded into a string using protobuf. The destination onion URL is acquired from potator.database.OnionIPMapper.getOnionUrl(). If the destination onion URL is not found for the IP address, an OURP request is sent by calling potator.ourp.OnionUrlResolutionProtocol.sendRequest(). If the destination is found, the spore encoded string is sent to its destination by calling potator.tor.server.Server.sendSpore().

sendGreeting(destination)[source]¶

Sends an OURP greeting to destination

Parameters:destination (str) – The onion URL to send the greeting to

The OURP greeting packet is crafted and sent to the onion URL by calling potator.tor.server.Server.sendSpore().

The OURP greeting is sent in a loop until Potator receives an OURP greeting acknowledge.
